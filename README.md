# Notion Covers Repository

This repository contains a collection of custom cover images designed specifically for automating a personal Notion database with over 100 pages.

## Purpose

The covers in this repository are used to automatically assign visual headers to Notion pages based on their content categories. This automation helps maintain visual consistency across a large database while reducing manual work.

## Structure

```
covers/
├── Ciencias de la tierra/    # Earth Sciences covers
└── Tecnologia/              # Technology covers
```

## Categories

### Ciencias de la tierra (Earth Sciences)
- Geology, Petroleum Engineering, and related fields
- 16 specialized covers for technical topics

### Tecnologia (Technology)  
- AWS services, Python, Machine Learning, and tech topics
- 15 covers for various technology domains

## Usage

These images are integrated with Notion's API to automatically:
- Assign appropriate covers based on page content
- Maintain visual consistency across 100+ database entries
- Reduce manual cover selection time

## Image Format
- PNG format for optimal quality
- Optimized for Notion's cover dimensions
- Consistent styling across all categories
